# 红楼梦儿童版
# Dream of the Red Chamber - Children's Edition

专为儿童改编的《红楼梦》故事，用简单有趣的语言讲述这部经典名著。

## 关于本书

《红楼梦》是中国四大名著之一，讲述了贾宝玉和他的家人、朋友们在大观园里的故事。

这个儿童版用简单易懂的语言，让小朋友也能了解这个美丽的故事。
